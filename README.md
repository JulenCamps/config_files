# Qtile installation

***Prerequisites***

Before installing Qtile you need to install **[xorg](https://wiki.archlinux.org/index.php/Xorg) to enable graphical interface:

```bash
sudo pacman -S xorg
```

***Installation***

Now you can install Qtile and the basic software that you will need:

Qtile:

```bash
sudo pacman -S Qtile    
```
